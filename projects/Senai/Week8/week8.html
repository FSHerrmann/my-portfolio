Week 8 – Senai Web Development Course

Overview

This folder contains the React/TypeScript digital-menu project for Week 8 of the Senai Web Development course. We built a fully componentized menu for “Mamamia Massas,” from navigation to footer.

Tasks

✅ Task 1: Digital Menu – Menu

Description: Create a React menu component with the restaurant name “Mamamia Massas” and navigation options: Home, Gnocchi, Pastas, Drinks.

✅ Task 2: Digital Menu – Presentation Screen

Description: Build a split-screen presentation component: left side with the slogan “Serving pasta for over 70 years” and sub-phrase “quality passed down through generations”; right side with a placeholder image of the restaurant façade.

✅ Task 3: Digital Menu – Gnocchi

Description: Implement two components: Gnocchi (4 options: Potato, Spinach, Carrot, Beetroot) as 2×2 cards displaying name, price, and image; and Sauces (Alla Puttanesca; Al Pesto di Basilico; Cacio e Pepe; Alla Matriciana) as an unordered list.

✅ Task 4: Digital Menu – Pastas

Description: Create cards (3 per row) for six pastas—Spaghetti, Tagliatelle, Fettuccine, Penne, Rigatoni, Pappardelle— each showing name, image, price, and allowed sauces via the Sauces component.

✅ Task 5: Digital Menu – Drinks

Description: Build a drinks section with categories: Water; Sodas (Coca-Cola, Guaraná, Sprite); Juices (Strawberry, Orange, Grape), each item displaying its price.

✅ Task 6: Digital Menu – Finalization

Description: Add a footer with social-media icons and a fictitious address, integrate all components into a master Menu (Cardápio) component, and render it in App.tsx with proper TypeScript typings.

📁 Located in: mamamia-menu
