Overview

This page contains all the programming exercises and logic problems completed in Week 4 of the Senai Web Development course. Each task is described below and implemented in JavaScript.

Tasks

✅ Task 1: Count Prime Numbers

Description: Develop a function that receives an integer and returns how many prime numbers exist up to and including that number. Use a for loop to solve this task.

Example: Input 10 → Result: 4 (prime numbers are 2, 3, 5, 7)

✅ Task 2: Count Even Numbers

Description: Create a function that receives an integer and returns how many even numbers exist from 1 up to that number (excluding 0). Use a while or do...while loop.

Example: Input 10 → Result: 5 (even numbers are 2, 4, 6, 8, 10)

✅ Task 3: Count Odd Numbers

Description: Create a function that receives an integer and returns how many odd numbers exist from 1 up to that number. You may use any loop structure.

Example: Input 10 → Result: 5 (odd numbers are 1, 3, 5, 7, 9)

✅ Task 4: Arrow Functions for Arithmetic

Description: Use arrow function syntax to create two functions: one for adding and one for subtracting two numbers.

✅ Task 6: Code Explanation – Even and Prime Numbers

Description: Provide a detailed explanation of the logic used to count even and prime numbers.

Even Numbers

The countEven function uses a while loop that starts at the input number and counts down to 1. In each iteration, it checks if the number is divisible by 2 (i.e., even). If so, it increases the counter. The final result is the total number of even integers found in the range.

Prime Numbers

The countPrimes function uses a for loop that iterates from the given number down to 2. For each number, it checks if it is divisible only by 1 and itself by testing all values from 2 up to that number (exclusive). If no divisor is found, it is counted as a prime. The function returns both the list of prime numbers found and their total count.
